ASCII Maze Generator
======
Prints out an ASCII maze to the terminal. The maze is generated using [Randomized Depth-First Search](https://en.wikipedia.org/wiki/Maze_generation_algorithm)

This program was written to be added to other projects and will generate a maze of any width and height, and returns that maze as a string.

## Example mazes
5x5
```
╺━┳━━━━━┓
╻ ┃ ╻ ╺━┫
┃ ┣━┻━╸ ┃
┃ ╹ ┏━╸ ╹
┗━━━┻━━━╸
```
